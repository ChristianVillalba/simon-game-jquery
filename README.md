# simon-game-jquery
Simon Game  this project uses JavaScript and the jQuery library

Warning: Sounds
How to play:
路 Simon will give the first signal.
路 Repeat the signal by pressing the same color button.
路 On each new level Simon will add a signal and
we have to repeat all the previous sequence adding the last signal.
路 If you fail to repeat the sequence exactly, the game is over.

Objective: Create a game with sounds and animations
